Question-1: What are the different ways to select an element in the DOM?

There are several ways to select an element in the DOM (Document Object Model) using JavaScript. Each method has its own use case depending on what kind of element you're trying to access and how specific or broad your selection needs to be.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Great question! Understanding the differences between innerHTML, innerText, and textContent is important when working with the DOM. These properties are used to get or set the content of elements, but they behave differently in terms of HTML parsing, performance, and visibility rules.

Question-3: What is event delegation in the DOM?

Event delegation in the DOM is a technique where instead of attaching event listeners to individual child elements, a single event listener is attached to a common parent element. This works because events in the DOM bubble up from the target element to its ancestors, allowing the parent to "catch" events from its children.

Question-4: What is event bubbling in the DOM?

Event bubbling in the DOM is a behavior where an event triggered on a child element propagates upward through its ancestors in the DOM tree.
